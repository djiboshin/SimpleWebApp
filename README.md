# SimpleWebApp
fast api + svelte

## Fast API:

1. create python venv
1. install fastapi
    ```
    pip install fastapi
    pip install "uvicorn[standard]"
    ```
1. start with `uvicorn backend.__main__:app --reload`

Poetry can also be used! 

## Svelte

1. Install node.js (**with npm!**)
2. `npm create svelte@latest .`
3. `npm install`
4. start with `npm run dev`
